arrayToCsv

(
  • a
)
String public static

Defined in source\core.js:60

Converts an array into a Comma Separated Values string. Each item in the array should be an array that represents one row in the CSV. Nulls and undefined values are interpreted as empty fields.

Parameters:

  • a String

    The array to convert

Returns:

String:

A CSV representation of the provided array.

Example:

var csv,
  books = [
    ['JavaScript: The Good Parts', 'Crockford, Douglas', 2008],
    ['Object-Oriented JavaScript', 'Stefanov, Stoyan', 2008],
    ['Effective JavaScript', 'Herman, David', 2012]
  ];

csv = CSV.arrayToCsv(books);

// csv now contains:
//
// JavaScript: The Good Parts,"Crockford, Douglas",2008\n
// Object-Oriented JavaScript,"Stefanov, Stoyan",2008\n
// Effective JavaScript,"Herman, David",2012\n

Defined in source\core.js:111

Converts a Comma Separated Values string into a multi-dimensional array. Each row in the CSV becomes an array. Empty fields are converted to nulls and non-quoted numbers are converted to integers or floats.

Parameters:

  • s String

    The string to convert

  • [config] Object optional

    Object literal with extra configuration. For historical reasons setting config to true is the same as passing {trim: true}, but this usage is deprecated and will likely be removed in the next version.

    • [trim=false] Boolean optional

      If set to True leading and trailing whitespace is stripped off of each non-quoted field as it is imported

Returns:

Array:

The CSV parsed as an array

Example:

var books,
  csv = 'JavaScript: The Good Parts,"Crockford, Douglas",2008\n' +
    'Object-Oriented JavaScript,"Stefanov, Stoyan",2008\n' +
    'Effective JavaScript,"Herman, David",2012\n';

books = CSV.csvToArray(csv);

// books now equals:
// [
//   ['JavaScript: The Good Parts', 'Crockford, Douglas', 2008],
//   ['Object-Oriented JavaScript', 'Stefanov, Stoyan', 2008],
//   ['Effective JavaScript', 'Herman, David', 2012]
// ];

Available since 1.2.0

Converts a Comma Separated Values string into an array of objects. Each row in the CSV becomes an object with properties named after each column. Empty fields are converted to nulls and non-quoted numbers are converted to integers or floats.

Parameters:

  • s String

    The string containing CSV data to convert

  • config Object

    Object literal with extra configuration

    • [columns] Array optional

      An array containing the name of each column in the CSV data. If not provided, the first row of the CSV data is assumed to contain the column names.

    • [trim] Boolean optional

      If true any field parsed from the CSV data will have leading and trailing whitespace trimmed

Returns:

Array:

The CSV parsed as an array of objects

Example:

var books,
  csv = 'title,author,year\n' +
    'JavaScript: The Good Parts,"Crockford, Douglas",2008\n' +
    'Object-Oriented JavaScript,"Stefanov, Stoyan",2008\n' +
    'Effective JavaScript,"Herman, David",2012\n';

books = CSV.csvToObject(csv);

// books now equals:
// [
//   {
//     title: 'JavaScript: The Good Parts',
//     author: 'Crockford, Douglas',
//     year: 2008
//   },
//   {
//     title: 'Object-Oriented JavaScript',
//     author: 'Stefanov, Stoyan',
//     year: 2008
//   },
//   {
//     title: 'Effective JavaScript',
//     author: 'Herman, David',
//     year: 2012
//   }
// ];

objectToCsv

(
  • arr
  • config
)
String static

Defined in source\core.js:291

Available since 1.2.0

Converts an array of objects into Comma Separated Values data Each propery on the objects becomes a column in the CSV data.

Parameters:

  • arr String

    An array of objects to be converted into CSV

  • config Object

    Object literal with extra configuration

    • [columns] Array optional

      An array containing the name of each column in the CSV data. If not provided, the column names will be inferred from the property names of the objects. Explicitly defining column names has several advantages:

      • It is faster since all column names are already known.
      • It allows you to specify a subset of the properties to use if you wish to.
      • It allows you to control what order the columns are output in, since the for...in statement used to infer field names does not guarantee a specific order.
    • [includeColumns=true] Boolean optional

      By default objectToCsv outputs the column names as the first row of the CSV data. Set to false to prevent this.

Returns:

String:

CSV data, each row representing an object from the input array, each field representing a property from those objects

Example:

var csv,
  books = [
    {
      title: 'JavaScript: The Good Parts',
      author: 'Crockford, Douglas',
      year: 2008
    },
    {
      title: 'Object-Oriented JavaScript',
      author: 'Stefanov, Stoyan',
      year: 2008
    },
    {
      title: 'Effective JavaScript',
      author: 'Herman, David',
      year: 2012
    }
  ];

csv = CSV.objectToCsv(books);

// csv now contains:
//
// title,author,year\n
// JavaScript: The Good Parts,"Crockford, Douglas",2008\n
// Object-Oriented JavaScript,"Stefanov, Stoyan",2008\n
// Effective JavaScript,"Herman, David",2012\n
